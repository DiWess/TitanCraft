# TitanCraft V1 Beta Blender Sources

Blender is not installed in this execution container, so binary `.blend` sources were not generated here. The exported `.gltf` candidates in `assets/models/v1_beta/` are deterministic, repo-owned blockout assets sized at 1 Blender/Godot unit = 1 meter and documented for later Blender authoring under the recommended filenames.
