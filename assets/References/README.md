# TitanCraft References

This directory is reserved for reference-only visual material used by future art-direction reviews.

- `Approved/` is for human-approved reference examples or text notes.
- `Rejected/` is for rejected reference examples or text notes.
- `Rejected/TitanCraftFailures/` is for TitanCraft-specific failure references.

No production art, Stage A art, generated Blender assets, or review PNGs are added by the Art Taste Pack. Any future binary or image reference must include source URL, license, author, date, hash, and classification before use.
