## Record Maintenance

This document is the source of truth for TitanCraft asset provenance. Update this record when:

- New assets are added to the project
- Assets are reclassified (e.g., PLACEHOLDER → PRODUCTION)
- Asset files are modified or replaced

Modifications to this record must include:

- Date of change
- Brief description of what changed
- Rationale (e.g., "Upgraded to final visual assets", "Removed deprecated placeholder")
- Link to related PR or issue if applicable
