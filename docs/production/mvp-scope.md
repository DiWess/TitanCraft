# Locked MVP Scope

TitanCraft MVP scope is **Crash Site**: one short, offline, Windows-first FPS survival/progression loop.

## Required MVP contents

- One crash site.
- FPS player.
- Resource pickups.
- Workbench craft.
- Mechanical arm.
- One Galaxabrain Scout.
- Retrieve component objective.
- Save point.
- Beacon.
- Victory, defeat, and save loop.
- Windows export.

## Locked loop

1. Spawn near the crash site.
2. Identify the next objective.
3. Collect the required resources.
4. Return to the workbench.
5. Craft the mechanical arm.
6. Fight one Galaxabrain Scout.
7. Retrieve the required component.
8. Activate the beacon.
9. Reach victory, defeat, or saved-state continuation.

## Forbidden expansions

- No extra biomes.
- No extra enemy families.
- No base-building expansion.
- No multiplayer.
- No broad inventory rewrite.
- No cinematic system.
- No lore expansion beyond MVP needs.
- No grappling hook, wall running, vehicles, complete rocket, large mech, cloud services, or remote telemetry.
