# Pull Request Review Checklist

- [ ] `README.md` was checked for scope conflicts.
- [ ] No forbidden MVP feature was added.
- [ ] Production scenes and gameplay files were not modified unless explicitly required.
- [ ] Tests or checks are appropriate for changed files.
- [ ] Evidence includes exact commands and outcomes.
- [ ] Manual test procedure is documented.
- [ ] Risks and limitations are documented.
- [ ] Final verdict uses approved vocabulary.
